wp-block-details
What is the WordPress REST API?

The WordPress REST API provides an interface for applications to interact with your WordPress site by sending and receiving data as JSON objects. It's the foundation of headless WordPress.

Why use headless WordPress?

Headless WordPress separates the content management backend from the front-end presentation layer, giving you freedom to build with any framework while still using WordPress as your CMS.

Dark mode

Dark mode styles are applied automatically via @media (prefers-color-scheme: dark). Toggle your OS dark mode to verify code blocks, borders, table headers, and caption colors adapt correctly.

// Code block — should use dark background in both modes
const mode = window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
Print

Use File → Print (or Ctrl/Cmd+P) to verify print styles: embeds are replaced with a placeholder, buttons and social icons are hidden, float alignments are cleared, and links display their URLs.
